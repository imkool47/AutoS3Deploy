CI/CD Static Website Deployment

Project Overview

This project demonstrates the automation of deploying a static website from a GitHub repository to an AWS S3 bucket using GitHub Actions.

Features

  • Automated deployment pipeline
  • Version control with GitHub
  • Hosting on AWS S3
  • Simple and elegant design

Technologies Used

  • HTML, CSS
  • GitHub Actions
  • AWS S3
